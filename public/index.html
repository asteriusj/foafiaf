FOAFIAF: Friend Of A Friend Is A Friend

What, Why, How - Value to community

A Rockford IL regional community network mapping capability leveraging RDF Linked Datasets for Social Graph visualization, analysis and question answering. The project data consists of semantic linked data strcutures reprenting ten entity types. These include Persons, Organizations, Places, Roles, Events, Notes and Concepts such as Strategies, Projects and Measures.

These entites, or Things, are the objects that make up a Social Graph that can depict personal and organizational relationsships.

The entity data is extensible and can be enhanced via Open Linked Data reposities. There are currenly over 2500 entity nodes that are linked with over 8000 property and object edges. Nodes and edges are the elements on graph data models on which FOAFIAF is implemented. Entity or Things are represented as graph nodes The tools are available at http://foafiaf.transformrockford.org/

FOAFIAF is a nod to Friend of a Friend, or acronym FOAF (ontology).
